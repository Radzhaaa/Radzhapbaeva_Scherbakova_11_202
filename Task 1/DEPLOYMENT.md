# Как запустить (задание 1)

У меня стоит питон 3, больше ничего не ставила.

1. Открыть папку Task 1 в терминале (или перейти в неё через cd).
2. Если файла urls.txt нет — сначала запустить: python get_urls.py. Он создаст список ссылок с википедии.
3. Потом запустить: python crawler.py. Страницы сохранятся в папку pages, index.txt создастся в Task 1.

Если какая-то ссылка не открылась — скрипт выведет сообщение и продолжит. Чтобы выкачать 100 страниц, в urls.txt нужно достаточно рабочих ссылок (я делала 100+ с вики).

Для сдачи: заархивировать папку pages и файл index.txt (оба лежат в Task 1 после запуска crawler.py) и отправить архив вместе со ссылкой на репо.
